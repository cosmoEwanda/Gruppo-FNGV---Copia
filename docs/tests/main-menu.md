# Main Menu tests

## Resolution check
1. Open the `Assets/Scenes/GameScenes/MainMenu` scene and hit the play button from the Unity Editor.
2. Click on 'Free Aspect' in the Unity Editor and change to a different resolution.
3. The UI components should stay in the same position.

## Game Mode selection
1. Open the `Assets/Scenes/GameScenes/MainMenu` scene and hit the play button from the Unity Editor.
2. Click on the `Gioca` button.
3. An overlay with the available game modes should appear.

## Tutorial Button
1. Open the `Assets/Scenes/GameScenes/MainMenu` scene and hit the play button from the Unity Editor.
2. Click on the `Tutorial` button.
3. The `Assets/Scenes/GameScenes/TutorialScene` scene should be loaded.
