# Gruppo FNGV

# Unity Editor Version: 6000.3.11f1 (LTS)

## Documentation
This project uses **MkDocs** for its documentation.

### Setup
To install the necessary dependencies, run:
```bash
pip install mkdocs
```

### Local Development
To launch the documentation server locally:
```bash
mkdocs serve
```
Once started, you can view the docs at `http://localhost:8000`.
